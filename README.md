# 🎬 Netflix Clone

A responsive Netflix-inspired web application built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of Netflix's homepage with a clean UI and responsive design.

---

## 🚀 Features

- Responsive design for different screen sizes
- Netflix-inspired homepage layout
- Interactive navigation bar
- Hero banner section
- Movie and TV show cards
- Hover effects and smooth UI interactions
- Clean and modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html**
   **OR**
4. Open it using **Live Server** in VS Code.
